# Frontend — Multi-Currency Payment

## Goal
UI bonita e funcional para employee e finance

## Out of scope

## Dependencies
Spec 06, Spec 01

## Decisions (fixas)
- Inertia + React no mesmo projeto Laravel
- Autenticação via localStorage + Authorization: Bearer
- Tailwind CSS
- Páginas mínimas:
    - Login/Register
    - Listagem com filtro por status
    - Detalhe
    - Criar request
    - Finance aprovar/rejeitar

## Deliverables
- Páginas React + Tailwind
- Proteção de rotas por auth/role (na UI e no backend)
- Testes simples de render

## Acceptance criteria (checável)
[ ] Rotas protegidas por autenticacao
[ ] Páginas implementadas usando Inertia e React


## DoD
[ ] Fluxo completo operável via UI
